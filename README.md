# Ansible devbox
Use Ansible to configure a developer system

## OS
- Debian 10+

## Prerequisites
Install the system prerequisites
```sh
$ ./run.sh prerequisites
```

## Usage
Run the Ansible playbook.
```sh
$ ./run.sh playbook
```
